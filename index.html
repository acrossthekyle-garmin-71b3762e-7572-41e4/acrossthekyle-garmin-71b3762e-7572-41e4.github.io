Barometer Widget

This widget shows the current pressure using the built-in sensor, along with a historical trend shown via an arrow below the gauge. A graph is also available showing the historical readings for the last X hours.

Compass (ABC) Widget

This widget was inspired by the ABC widget found on the Fenix watches, and it shows the current heading along with altitude, barometric pressure trend, altitude history, barometer history, current Lat/Long location along with GPS signal strength, and a dedicated compass screen for doing deviation.

Elevation Widget

This widget shows the current altitude using either the built-in sensor, or GPS. It will also show estimated Effective Oxygen at the given altitude. A graph is also available showing the data for the last X hours.

Purchase
Heart Rate Widget

This widget shows the current heart rate using the built-in sensor, the seven-day average resting heart rate, and an approximation of the HRV based on the heart beat intervals. A graph is also available showing the data for the last X hours.

Purchase
Sun Times Widget

This widget was inspired by the Sun Times widget found on Fenix watches. It will show the Sunrise/Sunset, along with Twilight, in a circle with color segments showing the various stages.

Purchase
Temperature Widget

This widget shows the current (approximate) external temperature using the built-in sensor. It will show "AIR TEMPERATURE" when not worn on the wrist, and "SKIN TEMPERATURE" when worn on the wrist. Core temperature is an estimation using the skin temperature reading. A graph is also available showing the data for the last X hours.
